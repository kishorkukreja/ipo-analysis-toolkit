# Scenario Method

## Two-Engine Model

Use both engines and reconcile them:

```text
Scenario target = fundamental value target - supply-overhang discount + event/catalyst adjustment
Fundamental value target = projected metric * scenario multiple
```

The 6-month range should emphasize listing-premium normalization, first results, anchor/pre-IPO unlocks, sector mood, and liquidity. The 12-month range should emphasize fundamentals, valuation, execution against IPO proceeds, and whether supply was absorbed.

## Primary Valuation Metric

| Issuer type | Primary target metric | Secondary check |
|---|---|---|
| Profitable consumer, IT services, pharma, diagnostics | EPS * P/E | EV/EBITDA, P/S |
| Manufacturing, logistics, hospitals, capital goods | EBITDA * EV/EBITDA | P/E, net debt |
| Banks/NBFC/HFC/MFI | Book value * P/B | ROA, ROE, credit cost |
| Life insurance | Embedded value * P/EV | VNB margin, APE growth |
| AMC/wealth | AUM or EBITDA multiple | P/E, market share |
| SaaS/platform/loss-making | Revenue, ARR, or gross-profit multiple | Burn, contribution margin |
| Real estate/infrastructure | NAV or pre-sales multiple | Net debt, collections |
| SME | Sector metric with liquidity discount | Traded value, lot size |

Do not compute P/E for loss-making issuers.

## Forecasting

If no guidance or analyst estimate exists:

```text
Projected metric 6m = latest annual or TTM metric * (1 + expected growth rate / 2)
Projected metric 12m = latest annual or TTM metric * (1 + expected growth rate)
```

Infer growth from RHP history, interim results, sector growth, order book/AUM/ARR, capacity, margins, and cash conversion. Haircut for one-customer growth, acquisition-led growth, weak OCF, working-capital stress, debt rise, capacity limits, or reversed sector trend.

## Multiple Selection

Start with peer median:

```text
Scenario multiple = peer median multiple * (1 + quality adjustment + growth adjustment + scarcity adjustment - risk adjustment - liquidity adjustment)
```

Suggested adjustment bands:

| Adjustment | Bull | Base | Bear |
|---|---:|---:|---:|
| Growth above peers | +10% to +30% | 0% to +15% | 0% |
| Margin/ROCE above peers | +5% to +20% | 0% to +10% | 0% |
| Category leader/scarcity | +5% to +20% | 0% to +10% | 0% |
| Rich IPO pricing | 0% to -10% | -10% to -25% | -25% to -50% |
| Weak governance/disclosure | -10% to -25% | -15% to -35% | -30% to -60% |
| SME liquidity | -10% to -25% | -15% to -35% | -25% to -50% |
| High supply overhang | -5% to -15% | -10% to -25% | -20% to -40% |

## Re-Rating and De-Rating

Re-rating drivers include earnings beats, margin expansion, order/AUM/ARR growth, cash conversion improvement, debt repayment, positive coverage, sector multiple expansion, institutional ownership, index eligibility, or orderly unlock absorption.

De-rating drivers include rich IPO pricing, earnings miss, growth slowdown, margin compression, weak OCF, delayed use of proceeds, related-party issues, litigation/regulatory risk, promoter pledge, aggressive unlock selling, sector compression, or SME volume collapse.

Explain multiple compression plainly:

```text
If EPS grows 15% but P/E compresses from 40x to 28x:
Target return roughly = 1.15 * (28 / 40) - 1 = -19.5%
```

## Return Bands

Use ranges, not point targets. Never show upside without downside.

| Post-listing setup | 6-month base range | 12-month base range |
|---|---:|---:|
| Listed flat/negative, fundamentals strong, valuation fair | 0% to +12% | +8% to +25% |
| Listed 10-25% premium, valuation fair | -5% to +10% | +5% to +20% |
| Listed 40%+ premium, valuation rich | -20% to +5% | -10% to +15% |
| Strong growth and early earnings beat | +10% to +30% | +20% to +50% |
| Earnings miss or lock-in overhang high | -15% to -35% | -20% to -45% |
| SME high-pop listing | -25% to +10% | -20% to +25% |
