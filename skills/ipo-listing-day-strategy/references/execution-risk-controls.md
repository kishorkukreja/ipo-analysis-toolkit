# Execution And Risk Controls

## Core Action Matrix

| Setup | Retail allottee action |
|---|---|
| IEP `>= +40%`, QIB strong, GMP confirmed | Sell 60-80%; trail rest with 8-12% stop from listing high or first-hour support. |
| IEP `+20% to +40%`, QIB strong, fair valuation | Sell 40-60%; hold 40-60% for 1-3 months only if first-hour price holds above open. |
| IEP `+10% to +20%`, QIB decent, fair valuation | Sell 25-50%; hold rest only if price holds above issue price and open. |
| IEP `+5% to +10%`, mixed subscription | Sell 50-70% if listing-gain objective; hold only with strong fundamentals. |
| Flat, QIB strong, valuation attractive | Hold initially with issue-price stop and 30-60 minute review. |
| Flat, QIB weak or valuation rich | Sell 70-100%; avoid waiting for rescue rally. |
| Negative IEP | Sell 80-100%; do not average down on listing day. |
| SME high GMP/large pop | Sell 70-90%; keep only token residual if fundamentals are exceptional. |
| SME flat/negative | Exit aggressively at reasonable liquidity; liquidity risk rises after initial window. |

## Stop-Loss Caveats

- Do not describe SPOS as a normal stop-loss environment. The pre-open is a call auction.
- In SPOS, plan with limit orders and broker-specific order behavior.
- After 10:00 AM, stops can be framed as:
  - Issue price breach.
  - Opening price break with volume.
  - First-hour low.
  - 5-8% below issue price for good mainboard fundamentals.
  - 3-5% below issue price for weak QIB or weak fundamentals.
- For SME negative listings, prioritize available bid depth over exact stop levels.

## SME Liquidity Warning

Always include for SME listings:

- SME lots can be large and harder to exit in one trade.
- Market-maker obligations improve quoting but do not guarantee execution at the displayed premium.
- Last traded price can overstate exitability if order depth is thin.
- Use limit orders and inspect bid depth before selling.
- High premium plus low volume is not the same as easy liquidity.

## Secondary Buying

Only discuss if asked. Default stance is conservative:

- Avoid buying in the first 5-15 minutes unless the user is an experienced trader.
- Do not chase vertical moves or low-float spikes.
- Require a base above open/VWAP and a defined stop.
- Keep intraday stops around 3-5% below entry or below first-hour support.
- Never use GMP as a post-listing buy target.

## Order Framing

- Prefer staged sells over all-or-nothing decisions unless the setup is clearly weak.
- Use limit orders in thin books and SME listings.
- Avoid market orders when displayed depth is small.
- If the user gives allotted quantity, convert sell/hold percentages into shares/lots where practical.
