# IPO Registrars Reference

Common Indian IPO registrars include KFin Technologies, MUFG Intime/Link Intime, Bigshare Services, Maashitla Securities, and issue-specific registrar portals named in the RHP/prospectus.

## Allotment Workflow

1. Identify the issue registrar from the RHP/prospectus, exchange page, or IPO notice.
2. Use the registrar's IPO allotment status page when active.
3. If registrar page is unavailable, use BSE/NSE allotment-status alternatives when applicable.
4. Ask for PAN, application number, or DP/client details only as needed, and do not request unnecessary sensitive information.

## Timeline Caveat

Default refund, mandate release, demat credit, and listing references should follow the current T+3 IPO cycle unless the issue is historical or officially follows another schedule.
