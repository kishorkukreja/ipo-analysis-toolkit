# Rubric Template

Each question passes only if the answer:

- Uses the intended workflow from `SKILL.md`.
- Includes required output sections from `assets/output-template.md`.
- Applies current Indian IPO rules correctly.
- Distinguishes mainboard vs SME where relevant.
- Uses source hierarchy and caveats correctly.
- Avoids unsupported investment advice.
- Handles missing data with explicit fallback behavior.

Critical failures:

- Presents T+6 as the current default IPO timeline.
- Treats GMP as official or sufficient for an apply decision.
- Gives guaranteed returns, allotment, or listing outcomes.
- Applies SME process guidance without checking issue date where the date matters.
